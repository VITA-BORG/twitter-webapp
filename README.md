# F3Ytwitter
This is a basic web app which interacts with the F3Y database

## Requirements
This application has a few requirements.  This project uses Go 1.18.
This application also requires 
- Go 1.18
- .env File Requirements
  - DB_USER
  - DB_PASS
  - DB_NAME
  - DB_HOST
  - DB_PORT
  - WEB_ADDR
  - BEARER_TOKEN
  - BEARER_TOKEN
  - SECRET_KEY
  - API_KEY
